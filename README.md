### 주요 내용
#### 일상생활에서 간편하게 접근이 가능한 저관여 웹앱
#### 다른 사이트에는 비슷한 분류로 이루어진 것을 볼 수 있음
#### 타사이트와 달리 여러분류가 포함이 되어있는 종합 저관여 웹앱

#### 쉬운 로그인으로 사용하는 종합 저관여 웹앱
#### 손쉬운 아이디찾기, 비밀번호 찾기
#### 일상에서 손쉽게 사용할 수 있는 저예산계산기, 체크리스트, 타이머 및 간단한 재밌거리
